# Database-Schemas
This repository contains SQL scripts converted to MySQL from the Murach repository, where the database was originally created in Microsoft SQL Server. It includes examples of table creation, primary and foreign keys, and data insertion to support practical learning in database courses.

**Reference:
**Pygator. (2015). SQL Server 2012/Databases/create_ap.sql. GitHub. https://github.com/Pygator/murach/blob/master/SQL%20Server%202012/Databases/create_ap.sql
